# PetCare宠伴 - 各端主菜单结构（v1.1）

## 一、C端：宠物主人端（小程序/App）

### 底部导航栏（5个Tab）

| Tab名称  | 图标 | 说明                           |
| -------- | ---- | ------------------------------ |
| **首页** | 🏠   | 核心功能入口 + 推荐内容        |
| **社区** | 💬   | 悬赏大厅 + 社区交流 + 宠物课堂 |
| **订单** | 📋   | 我的订单列表                   |
| **消息** | 🔔   | 系统通知 + 互动消息            |
| **我的** | 👤   | 个人中心 + 设置                |

---

### 1. 首页

**核心功能区**（双模式入口）：

- 🎯 **发布悬赏** - 自主定价，实名宠物主接单
- 📅 **预约服务** - 平台定价，认证宠托师接单

**推荐区域**：

- 热门悬赏订单（列表展示前5个）
- 精选社区动态（图文混排，前3条）

**Banner轮播**：

- 活动推广
- 新手引导
- 养宠知识推荐

---

### 2. 社区

**顶部Tab切换**（3个子Tab）：

#### 2.1 悬赏大厅

- 地图模式 / 列表模式切换按钮
- 筛选器：服务类型、时间范围、价格区间
- 排序：距离、金额、发布时间
- 订单卡片列表（显示：服务类型、时间、地点模糊、金额、宠物类型）
- **底部悬浮按钮**：**我要接单**（所有用户可见）
  - 已实名认证：点击提交接单意向
  - 未实名认证：点击弹出提示"请先完成实名认证"，提供"去认证"按钮，点击跳转实名认证页面

#### 2.2 社区交流

- 发布按钮（右下角悬浮）
- 动态流（下拉刷新、上拉加载）
- 动态卡片：发布者头像昵称、发布时间、图文/视频、点赞数、评论数、转发数
- 话题标签筛选：#猫咪日常 #养狗心得 #萌宠瞬间
- 互动操作：点赞、评论、转发、举报

#### 2.3 宠物课堂

- 分类Tab：喂养指南 / 健康管理 / 行为训练 / 疾病预防
- 文章/视频列表（封面图 + 标题 + 阅读量）
- 搜索框（搜索专业知识）
- 收藏入口

---

### 3. 订单

**顶部Tab切换**（4个子Tab）：

| Tab        | 说明                       |
| ---------- | -------------------------- |
| **全部**   | 所有订单                   |
| **待确认** | 已有人接单，等待宠物主确认 |
| **服务中** | 正在进行的服务             |
| **已完成** | 历史订单                   |

**订单卡片**（每个订单显示）：

- 订单编号
- 服务类型图标（喂养/遛狗/陪玩）
- 服务时间
- 宠物信息（名字、品种）
- 订单状态标签
- 金额
- 操作按钮：查看详情 / 取消 / 评价 / 投诉

**空状态**：

- 无订单时显示引导文案和"去预约"按钮

---

### 4. 消息

**消息分类Tab**：

| 分类         | 说明                         |
| ------------ | ---------------------------- |
| **订单通知** | 接单通知、服务提醒、完成通知 |
| **社区互动** | 点赞、评论、转发、关注通知   |
| **系统通知** | 平台公告、活动推送、违规警告 |
| **纠纷处理** | 投诉受理、裁决结果通知       |

**消息列表**：

- 未读红点标记
- 消息标题 + 摘要 + 时间
- 点击跳转到对应页面

**批量操作**：

- 一键已读
- 清空某类消息

---

### 5. 我的

**个人信息区**：

- 头像 + 昵称 + 会员等级
- 编辑资料入口

**功能菜单列表**：

| 模块         | 子项                                             |
| ------------ | ------------------------------------------------ |
| **我的宠物** | 宠物档案列表（最多5只），添加/编辑/删除          |
| **我的钱包** | 余额、积分、优惠券、充值入口                     |
| **地址管理** | 常用联系地址（最多3个）                          |
| **实名认证** | 认证状态、去认证入口                             |
| **我的收藏** | 收藏的动态、文章、宠托师                         |
| **我的关注** | 关注的用户列表                                   |
| **客服中心** | 常见问题、在线客服、投诉建议                     |
| **设置**     | 账号安全、隐私设置、通知设置、关于我们、退出登录 |

## 二、B端：服务提供者端（小程序/App）

### 底部导航栏（4个Tab）

| Tab名称  | 图标 | 说明                   |
| -------- | ---- | ---------------------- |
| **首页** | 🏠   | 今日概览 + 快速接单    |
| **订单** | 📋   | 订单列表 + SOP执行     |
| **收入** | 💰   | 收入明细 + 提现        |
| **我的** | 👤   | 认证状态 + 信用 + 设置 |

---

### 1. 首页

**顶部状态栏**：

- 认证状态标识（实名认证 / 认证宠托师）
- 信用分展示（默认100分，<80显示警告色）
- 接单开关（一键开启/关闭接单）

**今日数据卡片**：

- 今日订单数
- 今日收入
- 待处理事项（待接单、待服务、待评价）

**快速操作区**：

- 📍 **浏览悬赏** - 查看附近悬赏订单
- 📊 **收入报表** - 查看本周/本月收入趋势

** nearby订单推荐**：

- 附近悬赏订单列表（前5个）
- 显示：服务类型、时间、距离、金额
- 点击"我要接单"提交意向

---

### 2. 订单

**顶部Tab切换**（5个子Tab）：

| Tab        | 说明                       |
| ---------- | -------------------------- |
| **待接单** | 可接的悬赏订单 / 平台派单  |
| **待确认** | 已提交意向，等待宠物主确认 |
| **待服务** | 已确认，即将开始的服务     |
| **服务中** | 正在执行SOP的订单          |
| **已完成** | 历史订单                   |

**订单卡片**（根据状态显示不同操作）：

- **待接单**：查看详情 + 我要接单
- **待确认**：查看详情 + 取消意向
- **待服务**：查看详情 + 开始服务
- **服务中**：进入SOP执行页
- **已完成**：查看详情 + 查看评价

**空状态提示**：

- 无订单时显示鼓励文案

---

### 3. SOP执行页（从"服务中"订单进入）

**进度指示器**：

- 5步流程横向展示（进门消毒→拍照打卡→执行服务→清理现场→离开拍照）
- 当前步骤高亮，已完成步骤打勾

**每一步骤内容**：

#### 步骤1：进门消毒（预计2分钟）

- 操作指引文字
- 📷 拍摄门口照片按钮（含时间水印）
- 📷 拍摄消毒过程照片按钮
- ✅ 已完成消毒按钮

#### 步骤2：拍照打卡（预计3分钟）

- 操作指引文字
- 📷 拍摄宠物状态照片按钮（至少2张）
- 📝 记录精神状态下拉选择（活跃/平静/异常）
- ✅ 已完成打卡按钮

#### 步骤3：执行服务（根据类型不同）

- **上门喂养**：准备食物→喂食拍照→清洗食盆
- **遛狗**：佩戴牵引绳→出门拍照→遛狗过程拍照/视频→清理排泄物
- **陪玩**：玩具互动→互动拍照/视频→记录情绪变化
- 📷 上传照片/视频按钮
- ✅ 服务进行中按钮

#### 步骤4：清理现场（预计3分钟）

- 操作指引文字
- 📷 拍摄清理后环境照片按钮
- ✅ 已完成清理按钮

#### 步骤5：离开拍照（预计2分钟）

- 操作指引文字
- 📷 拍摄宠物离开状态照片按钮
- 📷 拍摄门锁后门口照片按钮
- 📝 填写服务小结文本框
- ✅ 服务完成按钮（所有步骤完成后才可点击）

**实时预览区**：

- 已上传的照片/视频缩略图
- 用户端同步可见

---

### 4. 收入

**收入概览卡片**：

- 今日收入
- 本周收入
- 本月收入
- 累计收入

**收入明细Tab切换**：

| Tab        | 说明             |
| ---------- | ---------------- |
| **全部**   | 所有收入记录     |
| **已结算** | 已到账的收入     |
| **待结算** | T+0结算中的收入  |
| **冻结中** | 纠纷中的订单收入 |

**收入列表项**（每条显示）：

- 订单编号
- 服务时间
- 金额
- 来源标签（悬赏/平台）
- 状态标签（已结算/待结算/冻结）

**提现功能区**：

- 可提现余额
- 💸 **立即提现**按钮
- 提现规则说明（最低10元，T+0到账，每月免费3次）
- 提现记录入口

**收入报表**：

- 周/月趋势图表
- 📥 导出Excel按钮

---

### 5. 我的

**个人信息区**：

- 头像 + 昵称
- 认证状态徽章（实名认证 / 认证宠托师）
- 综合评分（星级展示）
- 信用分（数字+进度条，<80红色警告）

**数据统计卡片**：

- 累计服务次数
- 好评率
- 复购率

**功能菜单列表**：

| 模块         | 子项                                             |
| ------------ | ------------------------------------------------ |
| **身份认证** | 实名认证状态、申请认证宠托师入口                 |
| **服务设置** | 可服务区域、可服务时间、最大同时订单数           |
| **我的评分** | 综合评分、各维度平均分、评价详情（最近50条）     |
| **信用记录** | 信用分、纠纷统计、最近10条纠纷记录               |
| **收入明细** | 跳转到收入页                                     |
| **提现记录** | 历史提现列表                                     |
| **客服中心** | 常见问题、在线客服、申诉入口                     |
| **设置**     | 账号安全、通知设置、隐私政策、关于我们、退出登录 |

## 三、后台管理系统（Web）

### 左侧导航菜单

#### 1. Dashboard（数据看板）

**核心指标卡片**：

- 今日活跃用户数
- 今日订单量
- 今日成交金额
- 今日新增用户

**趋势图表**：

- 近7日订单量趋势
- 近7日成交金额趋势
- 用户增长曲线

**待办事项**：

- 待审核宠托师数量
- 待处理投诉数量
- 待审核内容数量

**快捷操作**：

- 快速审核
- 查看订单
- 导出数据

---

#### 2. 用户管理

**2.1 用户列表**

- 搜索/筛选：用户名、手机号、注册时间、用户类型
- 表格列：用户ID、昵称、手机号、用户类型、注册时间、状态、操作
- 操作：查看详情、冻结/解冻、重置密码

**2.2 宠托师审核**

- Tab切换：待审核 / 已通过 / 已驳回
- 申请表详情：个人资料、身份证照片、微信支付分、考试成绩
- 操作：通过 / 驳回（需填写原因）

**2.3 信用管理**

- 用户信用列表
- 筛选：信用分区间、是否有纠纷
- 操作：查看详情、手动调整信用分、限制功能

---

#### 3. 订单管理

**3.1 订单列表**

- 搜索/筛选：订单号、用户、时间范围、订单类型、状态
- 表格列：订单号、订单类型、宠物主、服务提供者、服务时间、金额、状态、操作
- 操作：查看详情、手动取消、强制退款、导出

**3.2 纠纷处理**

- Tab切换：待处理 / 处理中 / 已结案
- 投诉详情：投诉类型、证据、双方陈述
- 裁决操作：支持宠物主 / 支持服务提供者 / 双方责任 / 证据不足
- 二次申诉处理入口

**3.3 订单统计**

- 按时间/类型/状态统计
- 导出Excel

---

#### 4. 内容管理

**4.1 社区动态审核**

- 待审核列表（系统识别疑似违规）
- 动态详情：内容、图片、发布者
- 操作：通过 / 拒绝（需填写原因）
- 批量操作

**4.2 宠物课堂管理**

- 文章/视频列表
- 操作：发布新内容、编辑、下架、置顶
- 数据统计：阅读量、点赞量、评论量

**4.3 举报处理**

- 用户举报的内容列表
- 查看详情、处理举报

---

#### 5. 财务管理

**5.1 收入明细**

- 平台总收入统计
- 按订单类型拆分（悬赏/平台）
- 导出财务报表

**5.2 提现审核**

- 待审核提现列表
- 审核操作：通过 / 驳回
- 提现记录查询

**5.3 退款管理**

- 退款申请列表
- 审核退款、手动退款
- 退款统计

**5.4 对账单**

- 生成日/周/月对账单
- 导出PDF/Excel

---

#### 6. 权限管理（RBAC）

**6.1 角色管理**

- 角色列表：角色名称、描述、关联用户数、状态
- 操作：创建角色、编辑、禁用/启用、复制、删除
- 预设角色：超级管理员、客服主管、普通客服、运营专员、财务专员、商家管理员

**6.2 权限点管理**

- 权限树展示（按模块分组）
- 模块：用户管理、订单管理、内容审核、财务管理、数据统计、系统设置、商家管理
- ��作：新增权限点、编辑、批量导入/导出
- 权限点详情：权限编码、名称、类型（菜单/按钮/API）

**6.3 角色权限分配**

- 选择角色 → 可视化权限树勾选
- 全选/全不选模块
- 保存后实时生效
- 权限对比功能（对比两个角色的权限差异）

**6.4 用户角色分配**

- 用户列表 + 角色分配弹窗
- 支持多角色分配
- 批量分配
- 显示用户完整权限列表

**6.5 审计日志**

- 日志列表：操作人、操作时间、操作IP、操作类型、变更内容
- 筛选：时间范围、操作人、操作类型
- 不可删除，只可归档
- 导出日志

---

#### 7. 数据统计

**7.1 运营数据看板**

- 用户数据：DAU、MAU、新增用户、留存率
- 订单数据：订单量、成交金额、转化率、客单价
- 区分悬赏订单和平台订单
- 服务提供者数据：数量、活跃度、平均评分

**7.2 社区数据**

- UGC内容量（动态数、评论数）
- 互动率（点赞率、评论率、转发率）
- 热门话题排行

**7.3 纠纷数据**

- 投诉率
- 裁决分布（支持宠物主/服务提供者/双方/证据不足占比）
- 平均处理时长
- 二次申诉率

**7.4 自定义报表**

- 选择指标、时间范围、维度
- 生成图表（折线图、柱状图、饼图）
- 导出Excel/PDF

---

#### 8. 系统设置

**8.1 SOP配置**

- 编辑5步SOP内容
- 设置每步预计时长
- 设置最少照片数量要求
- 违规处罚规则配置

**8.2 评分阈值**

- 暂停接单阈值（默认3.0星）
- 警告提示阈值（默认3.5星）
- 重新培训要求配置

**8.3 费率设置**

- 平台抽成比例
- 悬赏订单服务费
- 提现手续费

**8.4 通知模板**

- 编辑各类通知文案
- 推送渠道配置（站内信、微信模板消息）

**8.5 基础配置**

- 服务类型管理（喂养/遛狗/陪玩/医疗陪护）
- 宠物品种管理
- 地区管理

---

## 四、菜单权限映射示例

| 菜单项     | 所需权限编码             | 适用角色                     |
| ---------- | ------------------------ | ---------------------------- |
| Dashboard  | dashboard:view           | 所有管理员                   |
| 用户列表   | user:list                | 客服、运营、超级管理员       |
| 宠托师审核 | provider:approve         | 客服主管、超级管理员         |
| 订单列表   | order:list               | 客服、运营、财务、超级管理员 |
| 纠纷裁决   | dispute:resolve          | 客服、客服主管、超级管理员   |
| 内容审核   | content:audit            | 运营、超级管理员             |
| 提现审核   | finance:withdraw:approve | 财务、超级管理员             |
| 角色管理   | role:manage              | 超级管理员                   |
| 权限点管理 | permission:manage        | 超级管理员                   |
| 审计日志   | audit:view               | 超级管理员                   |
| 系统设置   | system:config            | 超级管理员                   |
