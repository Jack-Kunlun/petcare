PetCare Docker 部署架构

基于 Docker Compose 的多容器编排方案 · v1.0

核心服务

  • Admin: React + Nginx (Port 80)
  • Server: Nest.js API (Port 3001)
  • PostgreSQL 15 (Port 5432)
  • Redis 7 (Port 6379)

安全特性

  • Redis 密码认证 (--requirepass)
  • 端口暴露环境变量控制
  • CORS 动态配置
  • 生产环境禁用 Swagger UI
  • JWT 密钥强度验证 (≥32字符)

运维特性

  • Docker Volumes 数据持久化
  • 健康检查自动重启
  • 资源限制 (CPU/Memory)
  • 日志本地挂载便于调试
  • 启动依赖顺序控制
