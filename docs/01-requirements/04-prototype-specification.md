## 全局设计规范

### 色彩体系

| 用途     | 色值      | 说明                       |
| -------- | --------- | -------------------------- |
| 主色     | `#FF8C42` | 橙色，用于按钮/价格/选中态 |
| 背景色   | `#F5F5F5` | 浅灰，页面底色             |
| 卡片白   | `#FFFFFF` | 纯白，卡片背景             |
| 文字主色 | `#333333` | 深灰，正文                 |
| 文字次色 | `#999999` | 中灰，辅助信息             |
| 文字提示 | `#CCCCCC` | 浅灰，占位符               |
| 成功绿   | `#52C41A` | 状态标签/完成标记          |
| 警告红   | `#FF4D4F` | 删除确认/紧急标记          |

### 通用组件规范

- **圆角：** 卡片12px，按钮8px，头像圆形
- **阴影：** 卡片 `0 2px 8px rgba(0,0,0,0.08)`
- **间距：** 页面padding 16px，列表项间距12px
- **字体：** 标题加粗16-18px，正文14px，辅助12px
- **点击反馈：** 透明度0.9或缩放0.98

### 2. 悬赏大厅 `/rewards`

#### 2.1 列表模式（默认）

| 区块         | 内容                                         |
| ------------ | -------------------------------------------- |
| 顶部筛选栏   | 服务类型Tab + 排序下拉 + 筛选按钮            |
| 悬赏卡片列表 | 宠物头像+名称+服务内容+价格+距离+时间+发布人 |
| 悬浮发布按钮 | 右下角橙色"+"按钮                            | → `/rewards/publish` |

#### 2.2 地图模式 `/rewards?mode=map`

| 区块             | 内容                                         |
| ---------------- | -------------------------------------------- |
| 地图区域         | 全屏地图，价格气泡标记位置                   |
| 右上角计数       | "附近 N 个订单"标签                          |
| 右侧工具栏       | 放大/缩小/定位按钮                           |
| 底部面板（收起） | 宠物头像+名称+服务内容+价格+距离+时间+发布人 |
| 底部面板（展开） | 完整悬赏详情+操作按钮                        |

**底部面板交互：**

- 收起高度约120px，展示核心信息
- 点击任意位置→展开详情
- 左右箭头切换上一条/下一条
- 序号格式："当前/总数"（如"1/6"）

### 3. 发布悬赏流程

#### 步骤1：选择服务类型 `/rewards/publish/step1`

| 区块         | 内容                                                                                                                     |
| ------------ | ------------------------------------------------------------------------------------------------------------------------ |
| 顶部导航     | 返回箭头 + "发布悬赏" + 草稿按钮                                                                                         |
| 进度指示     | "步骤 1/3"                                                                                                               |
| 服务类型网格 | 6种服务卡片（2列）：上门喂养¥30-80、遛狗陪护¥40-100、上门洗澡¥80-150、宠物美容¥100-200、宠物医疗¥50-100、寄养照顾¥80-200 |
| 紧急程度     | 单选：普通/紧急/特急                                                                                                     |
| 下一步按钮   | 底部固定橙色按钮                                                                                                         | → step2 |

#### 步骤2：填写服务详情 `/rewards/publish/step2`

| 表单字段 | 类型                | 验证                 |
| -------- | ------------------- | -------------------- |
| 选择宠物 | 下拉/弹窗选择       | 必填，从"我的宠物"选 |
| 服务时间 | 日期+时间选择器     | 必填                 |
| 服务地址 | 省市区联动+详细地址 | 必填                 |
| 服务需求 | 多行文本            | 可选，最多200字      |
| 上传照片 | 图片上传（最多9张） | 可选                 |
| 预算范围 | 滑块/输入框         | 必填                 |

#### 步骤3：确认发布 `/rewards/publish/step3`

| 区块     | 内容                             |
| -------- | -------------------------------- |
| 预览卡片 | 宠物信息+服务内容+时间+地址+价格 |
| 附加选项 | 开关：需要保险/接受议价/允许代接 |
| 服务协议 | 勾选框+链接查看                  |
| 发布按钮 | 底部固定橙色按钮                 |

#### 发布成功页 `/rewards/publish/success`

| 区块     | 内容                      |
| -------- | ------------------------- |
| 成功动画 | ✓ 图标+动效               |
| 数据展示 | 预计响应时间/曝光量       |
| 后续操作 | "查看悬赏"/"返回首页"按钮 |

### 8. 宠物管理页面

#### 8.1 我的宠物列表 `/pets`

| 区块         | 内容                                            |
| ------------ | ----------------------------------------------- |
| 顶部导航     | 返回箭头 + "我的宠物" + 添加按钮(+)             |
| 宠物卡片网格 | 每行2列：头像+名称+品种+年龄/性别+编辑/删除图标 |
| 空状态       | 可爱插画+"暂无宠物，快去添加吧~"                |
| 底部悬浮按钮 | "+ 添加宠物"橙色主按钮                          |

**CRUD交互：**

- **创建：** 点击添加按钮→跳转添加页→填写表单→保存成功Toast
- **读取：** 点击卡片→跳转详情页
- **更新：** 点击编辑图标→预填充数据→修改后保存
- **删除：** 点击删除图标→确认对话框→确认后移除+Toast

#### 8.2 添加/编辑宠物页 `/pets/add` 或 `/pets/:id/edit`

| 表单字段  | 类型       | 验证                          |
| --------- | ---------- | ----------------------------- |
| 宠物头像  | 图片上传   | 可选，支持拍照/相册，实时预览 |
| 宠物名称* | 输入框     | 必填，最多10字符              |
| 品种*     | 下拉选择   | 必填，猫/狗/兔/仓鼠/鸟/其他   |
| 性别*     | 单选按钮组 | 必填，公/母/未知              |
| 出生日期  | 日期选择器 | 可选                          |
| 体重      | 数字输入   | 可选，单位kg，0.1-50范围      |
| 绝育状态  | 开关       | 已绝育/未绝育                 |
| 疫苗状态  | 多选       | 狂犬/猫三联/犬五联/其他       |
| 备注      | 多行文本   | 可选，最多200字               |

#### 8.3 宠物详情页 `/pets/:id`

| 区块       | 内容                                      |
| ---------- | ----------------------------------------- |
| 大尺寸头像 | 顶部居中，圆形80px                        |
| 基本信息   | 名称+品种标签+性别/年龄/体重/绝育状态网格 |
| 疫苗记录   | 列表带完成标记                            |
| 备注信息   | 如有则显示                                |
| 照片墙     | 横向滑动展示多张照片                      |
| 底部操作栏 | "编辑"按钮 + "删除"按钮                   |
